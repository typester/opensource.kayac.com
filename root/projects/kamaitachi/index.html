kamaitachi

perl flash media server.

Description

kamaitachi is perl implemantation of adobe's flash media server.

Currentry implemented several basic functions, such as:

  • remote procedure call (Remoting)
  • media streaming/recording from webcam

Download

kamaitachi is now on CPAN. So you can install it by running command:

cpan> install Kamaitachi

And development version and the sources are available at github:

git clone git://github.com/typester/kamaitachi.git
