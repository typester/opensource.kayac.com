marilena

actionscript3 object detection

Description

marilena は OpenCV の ActionScript3 移植プロジェクトです。

現在サポートしているのは以下の機能です。

  • OpenCV と共通の haar-cascade-xml の読み込み
  • 画像を読み込み、検出された顔の位置情報を flash.geom.Rectangle の配列で返す
