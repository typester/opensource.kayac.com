actionscript3 object detection

Description
-----------

marilena is a port of OpenCV Object Detetion to Actionscript3
currently implemented features:

 * reads the same haar-cascade-xml as OpenCV.
 * input an image, and outputs an array of flash.geom.Rectangle of face positions


Download
--------

marilena wiki at [SPARK Project, mash/Marilena](http://www.libspark.org/wiki/mash/Marilena) (Japanese)

source available:

    svn co http://www.libspark.org/as3/Marilena Marilena

sample:

[Marilena ObjectDetection in actionscript3 example: FaceDetection](http://maaash.jp/lab/Marilena_sample/)
